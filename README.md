# アテンディア - Attendiia

## どんなアプリ？

様々なコミュニティが広がる中で必要になる連絡。全員がLINEを持っていればいいけどやってない人だっている！
そんな時にSafariやGoogle、Microsoft Edge等のブラウザからアクセスできるアプリがこの「アテンディア」。

ぜひ一度使ってみてください！
